# Minimal Tabs for Zen Browser

A clean and highly compact tabs mod for **Zen Browser**.

### Features
- Very compact tab height (**26px**)
- Strong letter-spacing reduction for better readability
- Enhanced **Essential Tabs** (smaller + text hidden)
- Clean minimal look with Consolas font
- Lightweight and performance friendly

### Installation

**Easy way (Recommended):**
1. Download the latest release as ZIP.
2. Open Zen Browser → **Settings** → **Mods**
3. Click **Import** and select the downloaded ZIP file.
4. Restart Zen Browser.

**Manual way:**
1. Go to `about:support` and click **Open Folder** next to Profile Folder.
2. Create a folder named `chrome` (if it doesn't exist).
3. Copy `userChrome.css` into the `chrome` folder.
4. Restart Zen Browser.

> **Important**: Make sure `toolkit.legacyUserProfileCustomizations.stylesheets` is set to `true` in `about:config`.

### Customization
You can easily adjust:
- Tab height (`--tab-min-height`)
- Font size (`--tab-font-size`)
- Letter spacing (`.tab-label`)

### Compatibility
Tested on latest Zen Browser (2026).
